# 📚 EBookLand

## 프로젝트 소개
태블릿을 사용하는 현대 사회에 맞춰 다양한 형태의 책 서비스를 제공합니다.

- ✅ 전자책 구매 및 대여
- ✅ 전자책이 익숙하지 않은 분들을 위한 종이책 구매
- ✅ 공지사항 게시판 및 추천/비추천 기능
- ✅ 관리자와의 1:1 실시간 채팅

---

## 🛠 사용 기술

### Frontend
- HTML, CSS, JavaScript
- Bootstrap

### Backend
- Java, Spring Boot
- JPA (Java Persistence API)
- MySQL

### Tools & Environment
- IntelliJ IDEA
- GitHub

### Chat
- WebScket, Stomp
